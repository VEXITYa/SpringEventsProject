# Event Manager
## Описание
Этот проект — API-платформа для работы с мероприятиями и взаимодействия с участниками. Она состоит из двух отдельных сервисов, каждый из которых выполняет свою задачу.

Основной сервис предоставляет API для создания и редактирования мероприятий, управления списками участников и обработки регистраций. Встроенная авторизация и аутентификация обеспечивают безопасный доступ к функциям.

[Второй сервис](https://github.com/VEXITYa/SpringEventsNotificator) отвечает за уведомления. Он отслеживает изменения в мероприятиях и отправляет участникам сообщения об обновлениях, переносах и других важных изменениях. Благодаря этому пользователи всегда получают актуальную информацию о событиях, на которые зарегистрированы.

## Использованные технологии
- **Java 21**
- **Spring Boot 3.5**
- **Spring Web** — создание REST API
- **Spring Data JPA** — работа с базой данных
- **PostgreSQL** — основная СУБД
- **Spring Security** — авторизация и аутентификация
- **JWT (JJWT)** — управление токенами доступа
- **Spring Kafka** — обмен сообщениями между сервисами
- **Spring Validation** — проверка входных данных
- **Springdoc OpenAPI** — автоматическая документация API
